Getting Started

Welcome to the Dilla University Employee Management System. This guide will help you understand and effectively use the system's features.

1

System Requirements

Before using the system, ensure your device meets the following requirements:

  • Modern web browser (Chrome, Firefox, Safari, or Edge)
  • Stable internet connection
  • Minimum screen resolution: 1366 x 768
2

Logging In

To access the system:

  1. Open your web browser and navigate to the system URL
  2. Enter your username and password
  3. Click the "Login" button
Note: If you forget your password, use the "Forgot Password" link on the login page.
3

Dashboard Overview

The dashboard provides quick access to key features and important information:

  • Navigation sidebar for accessing different modules
  • Quick action buttons for common tasks
  • Summary cards showing important metrics
  • Recent activity feed
