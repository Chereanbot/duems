Leave Management

Learn how to manage employee leave requests, approvals, and leave policies in the system.

Leave Types

The system supports various leave types:

  • Annual Leave
  • Sick Leave
  • Personal Leave
  • Maternity Leave
  • Paternity Leave
  • Study Leave
  • Unpaid Leave
Note: Each leave type can have different approval workflows and documentation requirements.

Leave Policy Configuration

Configure leave policies for:

  • Leave allocation per employee
  • Carry-forward rules
  • Minimum notice period
  • Maximum consecutive days
  • Documentation requirements
  • Approval workflows
Warning: Changes to leave policies may affect existing leave balances and requests.

Managing Leave Requests

Steps for processing leave requests:

  1. Review the leave request details
  2. Check employee's leave balance
  3. Verify documentation (if required)
  4. Check for conflicts with other leaves
  5. Approve or reject the request
  6. Notify the employee of the decision
Tip: You can set up automatic notifications for pending leave requests and approvals.

Leave Reports

Available leave reports include:

  • Leave balance report
  • Leave utilization report
  • Department leave summary
  • Leave history report
  • Pending approvals report
Note: Reports can be filtered by date range, department, and leave type.

Leave Calendar

The leave calendar features:

  • Department-wide view
  • Individual employee view
  • Leave type color coding
  • Conflict detection
  • Export functionality
Tip: Use the calendar view to identify potential staffing gaps and plan accordingly.

Leave Analytics

The system provides analytics for:

  • Leave trends
  • Department comparison
  • Leave type distribution
  • Approval rates
  • Leave balance forecasting
Tip: Use analytics to optimize leave policies and improve workforce planning.
