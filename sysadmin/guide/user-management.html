User Management

Learn how to manage users in the system, including creating, editing, and deactivating user accounts.

Creating a New User

  1. Navigate to the Users section
  2. Click the "Add User" button
  3. Fill in the required information:
    • Personal details
    • Contact information
    • Department assignment
    • Role assignment
  4. Click "Save" to create the user
Tip: You can set a temporary password and require the user to change it on first login.

Managing User Accounts

Common user management tasks include:

  • Editing user information
  • Resetting passwords
  • Deactivating accounts
  • Managing user roles
Warning: Always verify user identity before performing sensitive operations.

User Search and Filtering

The system provides powerful search and filtering capabilities:

  • Search by name, email, or employee ID
  • Filter by department, role, or status
  • Sort by various fields
  • Export user lists for reporting

Bulk User Operations

For efficiency, you can perform operations on multiple users at once:

  1. Select multiple users using checkboxes
  2. Choose an operation from the bulk actions menu
  3. Confirm the operation
Note: Bulk operations are logged for audit purposes.
