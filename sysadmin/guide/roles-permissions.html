Roles & Permissions

Understand how to manage roles and permissions in the system.

Understanding Roles

The system includes predefined roles:

  • Administrator
  • Manager
  • Coordinator
  • Employee
  • Guest
Note: Each role has specific permissions that determine what actions users can perform.

Managing Permissions

To modify role permissions:

  1. Go to the Roles section
  2. Select the role to modify
  3. Click "Edit Permissions"
  4. Configure the desired permissions
  5. Save changes
Warning: Be cautious when modifying permissions, especially for administrative roles.

Creating Custom Roles

You can create custom roles to meet specific organizational needs:

  1. Navigate to the Roles section
  2. Click "Create New Role"
  3. Enter a name and description for the role
  4. Select the appropriate permissions
  5. Save the new role

Permission Categories

Permissions are organized into categories:

  • User Management
  • Department Management
  • Attendance Management
  • Leave Management
  • Report Generation
  • System Configuration
Tip: Use the "Copy Role" feature to create a new role based on an existing one.
