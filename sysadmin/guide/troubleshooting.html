Troubleshooting

Find solutions to common issues and learn how to resolve them.

Common Login Issues

If you're having trouble logging in:

  1. Check your internet connection
  2. Verify your username and password
  3. Clear your browser cache and cookies
  4. Try using a different browser
  5. Contact the system administrator if issues persist
Tip: Use the "Remember Me" option to avoid frequent logins.

System Performance Issues

If the system is running slowly:

  • Check your internet connection speed
  • Close unnecessary browser tabs and applications
  • Clear your browser cache
  • Try accessing the system during off-peak hours
Note: System performance may vary depending on the number of active users.

Report Generation Problems

If reports aren't generating correctly:

  1. Verify that you have the necessary permissions
  2. Check that your report parameters are correct
  3. Ensure there is data available for the selected time period
  4. Try generating a smaller report first
  5. Contact support if the issue persists

Mobile Access Issues

For problems accessing the system on mobile devices:

  • Ensure you're using a supported browser
  • Check your mobile data or Wi-Fi connection
  • Try clearing your browser cache
  • Update your browser to the latest version
Warning: Some features may have limited functionality on mobile devices.

Contacting Support

If you need additional assistance:

  1. Go to the Help Center
  2. Click on "Contact Support"
  3. Fill out the support request form with details about your issue
  4. Include any error messages or screenshots
  5. Submit the form and wait for a response
Note: Support is available during business hours (8:00 AM - 5:00 PM).
